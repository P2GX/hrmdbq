# Human Regulatory Mutation Database Curation Tool (HRMDBQ)


HRMDBQ is a tauri/angular desktop application that is designed to streamline annotation of non-coding Mendelian regulatory variants.

Predicting the consequences of mutations and distinguishing deleterious from neutral mutations in coding sequences are relatively straightforward thanks to the genetic code. The goal of the HRMD project is to comprehensivley curate the medical literature for Mendelian disease-causing non-coding variants as a foundation for analysis and classification tools.
